# 🌿 Jihad of Dawah — جهاد الدعوة بين عجز الداخل وكيد الخارج

**بِسْمِ اللَّهِ الرَّحْمَنِ الرَّحِيمِ**

Interactive trilingual (AR/EN/FR) educational web app based on the book **جهاد الدعوة بين عجز الداخل وكيد الخارج** by Sheikh Mohammed al-Ghazali (1917-1996).

## Features

- 20 cards on dawah as jihad, internal weakness, external plots, reform strategies, unity
- Trilingual: Arabic, English, French with RTL/LTR support
- 3 unique themes, responsive design + PWA-ready

## Tech

Pure HTML + CSS + JS. No frameworks. Just open `index.html`.

## Disclaimer

I am not a scholar. This is a humble effort to share Sheikh al-Ghazali's wisdom interactively. Not a fatwa.
